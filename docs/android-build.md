# Android Build Guide

## Prerequisites

- Java 17+ (JDK)
- Android Studio with Android SDK 34+
- Node.js 18+

## Debug Build

```bash
pnpm build
npx cap sync android
npx cap open android
```

In Android Studio: Build → Build Bundle(s) / APK(s) → Build APK(s)

## Development

With the dev server running (`pnpm dev`), the app loads against `http://localhost:5173`.

## Release Signing (Deferred)

Release signing is a pre-store task. To configure:

1. Generate a keystore:
   ```bash
   keytool -genkey -v -keystore release.keystore -alias reparilo -keyalg RSA -keysize 2048 -validity 10000
   ```
2. Add to `android/gradle.properties`:
   ```
   RELEASE_STORE_FILE=../release.keystore
   RELEASE_STORE_PASSWORD=...
   RELEASE_KEY_ALIAS=reparilo
   RELEASE_KEY_PASSWORD=...
   ```
3. Update `android/app/build.gradle` signingConfigs and buildTypes

## Notes

- `android/.gradle/`, `android/build/`, `android/app/build/`, `android/local.properties` are gitignored
- The scaffold was generated by `npx cap add android` and synced with `npx cap sync android`
