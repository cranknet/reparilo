# Code of Conduct

This project has adopted the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The full text is available at the link above. We adopt it by reference rather than reproducing it here, so that we automatically pick up any future corrections or clarifications published by the Contributor Covenant project.

A plain-text and translated versions are also available on the [Contributor Covenant website](https://www.contributor-covenant.org/translations/).

## Scope

This code applies to all project spaces, including:

- the GitHub repository (issues, pull requests, discussions, code review comments)
- any official communication channels the project may use in the future
- public spaces when an individual is officially representing the project

## Reporting

If you experience or witness behavior that violates the Contributor Covenant, please report it privately to:

**`security@reparilo.shop`**

All reports are handled confidentially. Please include:

- what happened
- when and where (link to issue, PR, or other context)
- any other people involved
- whether you've already taken any action

You can also report problems directly to GitHub using their [abuse reporting tools](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) if the behavior is on a GitHub surface.

## Enforcement

Reparilo is solo-maintained. The maintainer is responsible for reviewing reports and deciding on a response, which may include:

- a private warning
- a request to edit or delete content
- a temporary or permanent ban from the project's spaces
- escalation to GitHub's abuse team

The maintainer will follow the **Enforcement Guidelines** section of the Contributor Covenant 2.1 when deciding on consequences.

If a report involves the maintainer themselves, please contact GitHub directly using the link above — there isn't a separate appeal channel for a solo-maintained project, and the honest thing is to say so.
